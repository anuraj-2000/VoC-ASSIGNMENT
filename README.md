# Autonomous Voice of Customer Intelligence Agent

## Overview

This project builds an autonomous AI agent that continuously analyzes
public product reviews to generate actionable insights for product teams.

## Features

• Web scraping using Playwright  
• Sentiment + theme classification using Groq LLM  
• SQLite database storage  
• Duplicate detection  
• Weekly delta ingestion  
• Automated action reports  
• Conversational querying  

## Tech Stack

Node.js  
SQLite  
Playwright  
Groq LLM API  

## Setup

Install dependencies:

npm install

Add environment variables:

.env

GROQ_API_KEY=your_key

Run agent:

node agent/agent.js

Query insights:

node agent/queryAgent.js "What does Product B do better than Product A?"

## Initial Dataset

The repository includes a sample dataset generated by the agent during the first run.

The dataset is stored in:

database/reviews.db

It contains processed customer reviews including sentiment classification and thematic tagging.

This allows the system to be tested immediately without requiring a fresh scrape.

## Delta Ingestion Proof

The system performs incremental ingestion using a review hash to prevent duplicate entries.

A secondary execution of the agent produces the following output:

Duplicate review skipped
New reviews inserted: 0

This demonstrates that previously stored reviews are detected and skipped.
